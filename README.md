# DASHBOARD – WORKFORCE PERFORMANCE ANALYSIS

## Description
Dashboard developed in **Power BI** to analyze **workforce operational performance** within a law firm, covering multiple areas and roles across the organization.

The project included an **ETL process** to consolidate, clean, and structure operational data, enabling consistent evaluation of productivity and goal completion metrics.

## Analytical Focus
The dashboard provides a high-level view of workforce performance, allowing analysis at both individual and area levels, as well as monitoring process progress over time.

## General Metrics
- Volume of tasks / tasks handled  
- Process status (received, pending, completed)  
- Operational goal completion  
- Performance comparison across employees  
- Aggregated performance by area  

## Outcome
Enables monitoring of **employee productivity**, identification of performance differences, and supports management decisions based on reliable data.

## Tools
- Power BI  
- Power Query
